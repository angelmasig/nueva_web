Sesión 2

Realización de bitácora e intro al HTML

Marzo 22, 2023 Angel Maco Signori

Una bitácora cronológica en forma de blog utilizando HTML, que es un lenguaje de marcado que permite encerrar diferentes partes del contenido para que se vean o se comporten de una determinada manera. Se explican las etiquetas principales de HTML, como las de encabezados, listas, vínculos y párrafos, y se describen elementos como la etiqueta de apertura y cierre, los atributos y el anidamiento. Además, se mencionan los metadatos y la estructura básica de un documento HTML.
