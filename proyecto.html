Hydromatic

Sistema de riego con arduino

Ciclo 2023-1 Angel Maco Signori

Componentes: Arduino Uno: Microcontrolador que actúa como el cerebro del sistema. Sensor de humedad del suelo: Detecta la humedad del suelo para determinar cuándo regar las plantas. Bomba de agua: Suministra agua al sistema de riego. Relé: Actúa como interruptor para controlar la activación/desactivación de la bomba de agua. Válvula solenoide: Regula el flujo de agua hacia las plantas. Funcionalidades: Lectura del sensor de humedad: El Arduino lee los niveles de humedad del suelo utilizando el sensor. Verificación de la humedad del suelo: Compara los niveles de humedad con un valor umbral predefinido. Activación de la bomba de agua: Si el nivel de humedad está por debajo del umbral, el Arduino activa el relé para encender la bomba de agua. Riego de las plantas: La bomba de agua suministra agua a las plantas a través de la válvula solenoide. Desactivación del riego: Una vez que se alcanza el nivel de humedad adecuado, el Arduino desactiva la bomba de agua mediante el relé. Este sistema básico de riego con Arduino permite automatizar el proceso de riego de las plantas según sus necesidades de humedad del suelo, brindando una solución eficiente y conveniente para el cuidado de las plantas.
